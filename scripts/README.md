# Scripts

CI, benchmarking, and infrastructure setup scripts.

| Script | Purpose |
|--------|---------|
| `bench.sh` | Consolidated benchmark runner for Lambda Labs multi-GPU instances |
| `docker-cleanup.sh` | Docker container and image cleanup utilities |
| `run-multi-gpu-benchmark.sh` | Multi-GPU benchmark orchestration |
| `setup-lambda-benchmark.sh` | First-time Lambda Labs environment setup |

For runtime utilities (inspecting routes, managing backends, etc.), see `tools/`.
