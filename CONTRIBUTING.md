# Contributing

## Project Status

Early/experimental. This is a solo project and I'm still actively developing core functionality.

## Bug Reports

Welcome. Please open an issue with:
- Reproduction steps
- Environment details
- Relevant logs

## Feature Requests

Open an issue to discuss before implementing. I want to keep the scope focused.

## Pull Requests

I'm not currently accepting large PRs or new features from external contributors. Small fixes (typos, documentation, minor bugs) are welcome, but please open an issue first to confirm I'll accept it.

## Questions

Open a discussion or issue. I'll respond as time permits.
